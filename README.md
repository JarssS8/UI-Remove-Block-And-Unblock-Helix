# Simple UI to Remove, Block and Unblock Characters on Helix
## _Better and safe way to take this actions_

## Commands

- /CharRemoveBySteamID "SteamID"
- /CharBlockBySteamID "SteamID"
- /CharUnBlockBySteamID "SteamID"
